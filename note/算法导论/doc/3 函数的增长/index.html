# 算法导论3 函数的增长

算法的渐近（asymptotic）效率关注的是算法的运行时间如何随着输入规模的增长而增长。本章介绍了分析渐近效率时使用的数学符号和定义，并且给出了一些常用函数之间的渐近效率关系。

# 3.1 渐近符号

# Θ\ThetaΘ 符号

Θ\ThetaΘ 是希腊大写字母，英文写作 theta。对于给定函数 g(n)g(n)g(n)，Θ(g(n))\Theta(g(n))Θ(g(n)) 是一个函数集合：

Θ(g(n))={f(n)：存在正的常量 c1，c2 和 n0，使得任意 n⩾n0，满足 0⩽c1g(n)⩽f(n)⩽c2g(n)}\Theta(g(n)) = \{ f(n) \text{：存在正的常量 } c_1 \text{，} c_2 \text{ 和 } n_0 \text{，使得任意 } n \geqslant n_0 \text{，} \\ \text{满足 } 0 \leqslant c_1 g(n) \leqslant f(n) \leqslant c_2 g(n) \} Θ(g(n))={f(n)：存在正的常量c1​，c2​和n0​，使得任意n⩾n0​，满足0⩽c1​g(n)⩽f(n)⩽c2​g(n)}

如果 f(n)f(n)f(n) 满足条件，那么可以写作 f(n)∈Θ(g(n))f(n) \in \Theta(g(n))f(n)∈Θ(g(n))，不过通常会被写成 f(n)=Θ(g(n))f(n) = \Theta(g(n))f(n)=Θ(g(n))。此时，g(n)g(n)g(n) 被称作 f(n)f(n)f(n) 的一个渐近紧界（asymptotically tight bound）。

在该定义中，每一个 f(n)∈Θ(g(n))f(n) \in \Theta(g(n))f(n)∈Θ(g(n)) 都是渐近非负（asymptotically nonnegative）的。也就是说存在 n0n_0n0​ 使得任意 n⩾n0n \geqslant n_0n⩾n0​ 满足 f(n)⩾0f(n) \geqslant 0f(n)⩾0。g(n)g(n)g(n) 必须是渐近非负的，否则 Θ(g(n))\Theta(g(n))Θ(g(n)) 就是空集了。类似地，还有渐近正的定义：存在 n0n_0n0​ 使得任意 n⩾n0n \geqslant n_0n⩾n0​ 满足 f(n)>0f(n) > 0f(n)>0。

# OOO 符号

Θ\ThetaΘ 符号渐近地把函数约束在一个渐近上界和渐近下界范围内。当只有渐近上界（asymptotic upper bound）时，可以使用 OOO 符号。对于给定函数 g(n)g(n)g(n)，O(g(n))O(g(n))O(g(n)) 是一个函数集合：

O(g(n))={f(n)：存在正的常量 c 和 n0，使得任意 n⩾n0，满足 0⩽f(n)⩽cg(n)}O(g(n)) = \{ f(n) \text{：存在正的常量 } c \text{ 和 } n_0 \text{，使得任意 } n \geqslant n_0 \text{，} \\ \text{满足 } 0 \leqslant f(n) \leqslant c g(n) \} O(g(n))={f(n)：存在正的常量c和n0​，使得任意n⩾n0​，满足0⩽f(n)⩽cg(n)}

通常可以通过观察算法的整体结构直接得到渐近上界。举例来说，根据插入排序二层嵌套循环，可以直接得到它在最坏情况下具有 O(n2)O(n^2)O(n2) 的运行时间上界。

# Ω\OmegaΩ 符号

Ω\OmegaΩ 是希腊大写字母，英语写作 omega。类似地，Ω\OmegaΩ 只提供渐近下界（asymptotic lower bound）。对于给定函数 g(n)g(n)g(n)，Ω(g(n))\Omega(g(n))Ω(g(n)) 是一个函数集合：

Ω(g(n))={f(n)：存在正的常量 c 和 n0，使得任意 n⩾n0，满足 0⩽cg(n)⩽f(n)}\Omega(g(n)) = \{ f(n) \text{：存在正的常量 } c \text{ 和 } n_0 \text{，使得任意 } n \geqslant n_0 \text{，} \\ \text{满足 } 0 \leqslant c g(n) \leqslant f(n) \} Ω(g(n))={f(n)：存在正的常量c和n0​，使得任意n⩾n0​，满足0⩽cg(n)⩽f(n)}

# 定理 3.1

对于两个函数 f(n)f(n)f(n) 和 g(n)g(n)g(n)，f(n)=Θ(g(n))f(n) = \Theta(g(n))f(n)=Θ(g(n)) 当且仅当 f(n)=O(g(n))f(n) = O(g(n))f(n)=O(g(n)) 且 f(n)=Ω(g(n))f(n) = \Omega(g(n))f(n)=Ω(g(n))。

# ooo 符号

OOO 符号提供的渐近上界可能是不紧的。比如 2n2=O(n2)2 n^2 = O(n^2)2n2=O(n2) 是紧的，但是 2n=O(n2)2 n = O(n^2)2n=O(n2) 不是。因此使用 ooo 符号来特指不紧的渐近上界，其正式定义是：

o(g(n))={f(n)：对于任意正的常数 c>0，总存在一个常数 n0>0，使得任意 n⩾n0，满足 0⩽f(n)⩽cg(n)}o(g(n)) = \{ f(n) \text{：对于任意正的常数 } c > 0 \text{，总存在一个常数 } n_0 > 0 \text{，} \\ \text{使得任意 } n \geqslant n_0 \text{，} \text{满足 } 0 \leqslant f(n) \leqslant c g(n) \} o(g(n))={f(n)：对于任意正的常数c>0，总存在一个常数n0​>0，使得任意n⩾n0​，满足0⩽f(n)⩽cg(n)}

从上述定义可知，当 nnn 逐渐趋向无限大时，f(n)/g(n)f(n) / g(n)f(n)/g(n) 会越来越小（否则对于很小的 ccc 就无法找到 n0n_0n0​了）。因此有：

lim⁡n→∞f(n)g(n)=0\lim_{n \rightarrow \infty} \frac{f(n)}{g(n)} = 0 n→∞lim​g(n)f(n)​=0

也有某些教材把这个极限用作 ooo 符号的定义。

# ω\omegaω 符号

ω\omegaω 是希腊小写字母，英语写作 omega。与 ooo 符号类似，ω\omegaω 符号用于表示不紧的渐近下界。

ω(g(n))={f(n)：对于任意正的常数 c>0，总存在一个常数 n0>0，使得任意 n⩾n0，满足 0⩽cg(n)⩽f(n)}\omega(g(n)) = \{ f(n) \text{：对于任意正的常数 } c > 0 \text{，总存在一个常数 } n_0 > 0 \text{，} \\ \text{使得任意 } n \geqslant n_0 \text{，} \text{满足 } 0 \leqslant c g(n) \leqslant f(n) \} ω(g(n))={f(n)：对于任意正的常数c>0，总存在一个常数n0​>0，使得任意n⩾n0​，满足0⩽cg(n)⩽f(n)}

当 nnn 逐渐趋向无限大时，f(n)/g(n)f(n) / g(n)f(n)/g(n) 会越来越大。因此有：

lim⁡n→∞f(n)g(n)=∞\lim_{n \rightarrow \infty} \frac{f(n)}{g(n)} = \infty n→∞lim​g(n)f(n)​=∞

也有某些教材把这个极限用作 ω\omegaω 符号的定义。

# 函数比较

f(n)f(n)f(n) 渐近小于（asymptotically smaller）g(n)g(n)g(n) 当且仅当 f(n)=o(g(n))f(n) = o(g(n))f(n)=o(g(n))。f(n)f(n)f(n) 渐近大于（asymptotically larger）g(n)g(n)g(n) 当且仅当 f(n)=ω(g(n))f(n) = \omega(g(n))f(n)=ω(g(n))。

不是所有函数都是渐近可比较的。存在函数 f(n)f(n)f(n) 和 g(n)g(n)g(n)，既不满足 f(n)=O(g(n))f(n) = O(g(n))f(n)=O(g(n))，也不满足 f(n)=Ω(g(n))f(n) = \Omega(g(n))f(n)=Ω(g(n))。举个例子，f(n)=nf(n) = nf(n)=n，g(n)=n1+sin⁡ng(n) = n^{1 + \sin n}g(n)=n1+sinn。

# 3.2 常用函数

# 多项式

给定整数 ddd，ddd 阶多项式指的是函数 p(n)p(n)p(n)：

p(n)=∑i=0dainip(n) = \sum_{i = 0}^d a_i n^i p(n)=i=0∑d​ai​ni

其中，a0,a1,…,ada_0, a_1, \dots, a_da0​,a1​,…,ad​ 被称为系数（coefficient），并且 ad≠0a_d \ne 0ad​=0。多项式渐近正当且仅当 ad>0a_d > 0ad​>0。渐近正 ddd 阶多项式 p(n)p(n)p(n) 满足 p(n)=Θ(nd)p(n) = \Theta(n^d)p(n)=Θ(nd)。如果存在常量 kkk 使得 f(n)=O(nk)f(n) = O(n^k)f(n)=O(nk)，那么 f(n)f(n)f(n) 是多项式有界（polynomially bounded）的。

# 指数

对于所有的实数常量 aaa 和 bbb 且 a>1a > 1a>1，有：

lim⁡n→∞nban=0\lim_{n \rightarrow \infty} \frac{n^b}{a^n} = 0 n→∞lim​annb​=0

因此可以得到：

nb=o(an)n^b = o(a^n) nb=o(an)

综上，底数大于 111 的指数函数比任何多项式函数增长都快。

# 对数

如果存在常量 kkk 使得 f(n)=O(lg⁡kn)f(n) = O(\lg^k n)f(n)=O(lgkn)，那么 f(n)f(n)f(n) 是多对数有界（polylogarithmically bounded）的。对于上一小节中的极限，使用 lg⁡n\lg nlgn 替换 nnn，使用 2a2^a2a 替换 aaa（新的 aaa 的范围是 a>0a > 0a>0），可以得到：

lim⁡n→∞lg⁡bn(2a)lg⁡n=lim⁡n→∞lg⁡bnna=0\lim_{n \rightarrow \infty} \frac{\lg^b n}{(2^a)^{\lg n}} = \lim_{n \rightarrow \infty} \frac{\lg^b n}{n^a} = 0 n→∞lim​(2a)lgnlgbn​=n→∞lim​nalgbn​=0

因此可以得到：

lg⁡bn=o(na)\lg^b n = o(n^a) lgbn=o(na)

综上，正多项式函数比任何多对数函数增长都快。

# 阶乘

n!={1如果 n=0n(n−1)!如果 n>0n! = \begin{cases} 1 & \text{如果 } n = 0 \\ n (n - 1)! & \text{如果 } n > 0 \end{cases} n!={1n(n−1)!​如果n=0如果n>0​

n!=2πn(ne)n(1+Θ(1n))n! = \sqrt{2 \pi n} (\frac{n}{e})^n (1 + \Theta(\frac{1}{n})) n!=2πn​(en​)n(1+Θ(n1​))

综上，阶乘比任何底数大于 111 的指数函数增长都快，但是比 nnn^nnn 慢。
