# 算法导论8 线性时间排序

前几章中提到的几种排序方法，例如插入排序、归并排序、堆排序和快速排序，有一个共同点：顺序判断仅基于元素间的比较。这类排序算法称为比较排序（comparsion sort）。

接下来会先证明比较排序算法都是 Ω(nlg⁡n)\Omega(n \lg n)Ω(nlgn)，然后给出几个可以在线性时间内排序的算法。

# 最坏情况的下界

最坏的比较次数等于决策树的高度。因此所有决策树的高度下界（虽然叫下界，但是指的是最高的情况）就是所有比较排序的运行时间下界。

# 定理 8.1

比较排序算法在最坏情况下需要 Ω(nlg⁡n)\Omega(n \lg n)Ω(nlgn) 次比较。

假设排序 nnn 个元素时的决策树高度是 hhh，且共有 lll 个叶节点。注意，上文中给出的决策树例子是“最简”的，因为可以在 b:c 节点的子节点中再比较一次 a:b，这样决策树中就会多出许多无用的比较，从而产生很多重复的叶节点。nnn 个元素的排列一共有 n!n!n! 种，高度为 hhh 的二叉树叶节点不会超过 2h2^h2h 个，因此有不等式：

n!⩽l⩽2hn! \leqslant l \leqslant 2^h n!⩽l⩽2h

对不等式最左和最右同时取对数：

h⩾lg⁡(n!)h \geqslant \lg(n!) h⩾lg(n!)

回忆 Stirling 近似：

lg⁡(n!)=lg⁡(2πn(ne)n(1+Θ(1n)))=12lg⁡(2πn)+nlg⁡(ne)+lg⁡(1+Θ(1n))=Θ(nlg⁡n)\begin{aligned} \lg(n!) & = \lg(\sqrt{2 \pi n} (\frac{n}{e})^n (1 + \Theta(\frac{1}{n}))) \\ & = \frac{1}{2} \lg(2 \pi n) + n \lg(\frac{n}{e}) + \lg(1 + \Theta(\frac{1}{n})) \\ & = \Theta(n \lg n) \end{aligned} lg(n!)​=lg(2πn​(en​)n(1+Θ(n1​)))=21​lg(2πn)+nlg(en​)+lg(1+Θ(n1​))=Θ(nlgn)​

所以 h=Ω(nlg⁡n)h = \Omega(n \lg n)h=Ω(nlgn)。

# 推论 8.2

堆排序和归并排序是渐近最优的比较排序。

# 8.2 计数排序

计数排序（counting sort）假设输入的 nnn 个整数在区间 [0,k][0, k][0,k] 中，其中 kkk 是某个整数。当 k=O(n)k = O(n)k=O(n) 时，计数排序的运行时间是 Θ(n)\Theta(n)Θ(n)。计数排序的伪代码 COUNTING-SORT(A, B, k)（A 是待排序数组，B 是输出数组）如下：

COUNTING-SORT(A, B, k)
  创建新数组 C[0..k]（数组中元素默认值为 0）

  // 计数
  for j = 1 to A.length
    C[A[j]] = C[A[j]] + 1

  // 运行前：C[i] 中的值代表原数组中有多少等于 i 的元素
  // 运行后：C[i] 中的值代表原数组中有多少小于等于 i 的元素
  for i = 1 to k
    C[i] = C[i] + C[i-1]

  for j = A.length downto 1
    B[C[A[j]]] = A[j]
    C[A[j]] = C[A[j]] - 1

# 引理 8.3

给定 nnn 个 ddd 位的数字，每一位上可能出现 kkk 种不同的值。如果稳定排序算法的运行时间是 Θ(n+k)\Theta(n + k)Θ(n+k)，那么基数排序的运行时间是 Θ(d(n+k))\Theta(d(n + k))Θ(d(n+k))。

如果 ddd 是常数且 k=O(n)k = O(n)k=O(n)，那么基数排序的运行时间就是 Θ(n)\Theta(n)Θ(n) 了。

# 引理 8.4

给定 nnn 个 bbb 比特的数字和任意正整数 r⩽br \leqslant br⩽b，如果稳定排序算法的运行时间是 Θ(n+k)\Theta(n + k)Θ(n+k)，那么基数排序的运行时间是 Θ((b/r)(n+2r))\Theta((b/r)(n + 2^r))Θ((b/r)(n+2r))。

举个例子来说，可以把 323232 比特的数字看成四个 888 比特数字拼接而成。此时有 b=32b = 32b=32，r=8r = 8r=8。

# 8.4 桶排序

如果输入符合均匀分布，桶排序（bucket sort）的平均情况运行时间是 O(n)O(n)O(n)。

假设输入是在 [0,1)[0, 1)[0,1) 区间的实数。桶排序把 [0,1)[0, 1)[0,1) 区间分成 nnn 个相同大小的子区间，称为桶。排序时，桶排序把要排序的数字放入对应的桶中，然后把每个桶中的元素排序。

BUCKET-SORT(A)
  n = A.length
  创建新数组 B[0..n - 1]
  for i = 0 to n - 1
    B[i] 为空数组
  for i = 1 to n
    把 A[i] 插入到 B[n * A[i]] 中
  for i = 0 to n - 1
    使用插入排序把 B[i] 排序
  将 B[0] 到 B[n - 1] 连接起来
